A.7. index.cgi doesn't show up unless specified in the URL

You probably need to set up your web server in such a way that it will serve the index.cgi page as an index page.

If you are using Apache, you can do this by adding index.cgi to the end of the DirectoryIndex line as mentioned in Section 2.2.4.1, “Bugzilla using Apache”.
